Bulk Renaming Executable

Allows you to rename every file of a specific file type in a given directory.

Steps to run after download:

STEP 1: Move the exe to the directory with the files you want to rename.
 STEP 2: Open a command prompt and navigate to the folder with the files and the executable.
 STEP 3: Run the command: bulk_rename -bn {new_file_name} -x {file_type} -c {number or date ordering}
       - Example: bulk_rename -bn quarterly_accounts -x xlsx -c date
       - Example: bulk_rename -bn july_invoices -x docx -c number
 STEP 4: The file names will be changed!
