Installation Instructions

Step 1: Download the zip file

After downloading the zip file, locate it in your downloads folder.

Note: The extension folder needs to remain on your computer as long as you want to use the extension. Don't delete it after installation.

Step 2: Extract the contents of the zip

Right click the zip file and extract.

Step 3: Open Chrome Extensions Page

1) Open Google Chrome and type chrome://extensions in the address bar, then press Enter.

2) In the top-right corner of the Extensions page, toggle on "Developer mode".

3) Click the "Load unpacked" button in the top-left corner.

Step 4: Load your extension folder into chrome

Navigate to the folder where you extracted the zip file and select the folder (not the zip file itself).

The folder you select should be like "C:/Downloads/ChatCaptureDeploy1_1_6/ChatCaptureDeploy1_1_6"

Step 5: Verify Installation

You should now see "Interactive Contexto" in your list of extensions.

Step 6: Ready to Use!

You're now ready to use Interactive Contexto during your live streams.
