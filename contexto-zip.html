Installation Instructions

Step 1: Extract the Zip File

After downloading the zip file, locate it in your downloads folder and extract it to a location where you won't accidentally delete it.

Note: The extension folder needs to remain on your computer as long as you want to use the extension. Don't delete it after installation.

Step 2: Open Chrome Extensions Page

Open Google Chrome and type chrome://extensions in the address bar, then press Enter.

Step 3: Enable Developer Mode

In the top-right corner of the Extensions page, toggle on "Developer mode".

Step 4: Load Unpacked Extension

Click the "Load unpacked" button in the top-left corner.

Navigate to the folder where you extracted the zip file and select the folder (not the zip file itself).

Step 5: Verify Installation

You should now see "Chat Capture: Contexto" in your list of extensions. Make sure it's enabled by checking that the toggle is on.

Tip: Pin the extension to your toolbar by clicking the puzzle piece icon in Chrome's toolbar and then the pin icon next to Chat Capture: Contexto.

Step 6: Ready to Use!

You're now ready to use Interactive Contexto during your live streams. Go to any Contexto game page and the extension will automatically work.
