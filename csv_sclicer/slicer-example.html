Steps to run after download:

STEP 1: Move the file to a directory that only has excel files you want to edit.
 STEP 2: Open a command prompt and navigate to the folder with the excel files and the executable.
 STEP 3: Run the command (columns in file must not have spaces): csv_slicer -keep column1 column2 column3
 STEP 4: The new files will be added to your current directory and the old files will be added to a directory called "raw_files" next to the current folder
